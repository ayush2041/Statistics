# Statistics for Data Science, Machine Learning, and AI

Master the most important statistics concepts required for Data Science, Machine Learning, and AI.

This repository contains a collection of Jupyter Notebook tutorials covering essential statistical tests and concepts that form the foundation of data science and machine learning.

## 📚 Contents

### Hypothesis Testing

The repository covers the following statistical tests:

- **[t-test](t-test.ipynb)** - Single sample t-test for comparing sample means against population means
- **[Two Sample T-test](twoSample_T-test.ipynb)** - Compare means between two independent samples
- **[z-test](z-test.ipynb)** - Standard normal distribution test for large samples
- **[ANOVA Test](ANOVA-test.ipynb)** - Analysis of Variance for comparing multiple groups
- **[Chi-Square Test](ChiSquare-test.ipynb)** - Test for independence and goodness of fit for categorical data

## 🎯 Key Concepts Covered

These notebooks cover critical statistical concepts including:

- **Hypothesis Testing** - Formulating null and alternative hypotheses
- **Statistical Significance** - Understanding p-values and significance levels
- **Test Selection** - Choosing the right statistical test for your data
- **Assumptions & Validity** - Understanding test assumptions and limitations
- **Real-world Applications** - Practical examples in data science and ML

## 🚀 Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook
- Required libraries:
  - NumPy
  - Pandas
  - SciPy
  - Matplotlib
  - Seaborn

### Installation

```bash
# Clone the repository
git clone https://github.com/ayush2041/Statistics.git
cd Statistics

# Install required packages
pip install numpy pandas scipy matplotlib seaborn jupyter
```

### Usage

1. Start Jupyter Notebook:
```bash
jupyter notebook
```

2. Open any `.ipynb` file to explore the tutorials
3. Follow the explanations, code examples, and exercises in each notebook

## 📖 Learning Path

We recommend following this learning path:

1. Start with **z-test** - Understanding the basics of hypothesis testing
2. Move to **t-test** - Working with sample statistics
3. Explore **Two Sample T-test** - Comparing between groups
4. Learn **ANOVA Test** - Extending to multiple groups
5. Master **Chi-Square Test** - Categorical data analysis
